Coloca aquí el archivo icon.ico para la aplicación de Windows.

El ícono debe ser:
- Formato: .ico
- Nombre: icon.ico
- Tamaños recomendados: 16x16, 32x32, 48x48, 256x256

Puedes crear un ícono usando herramientas como:
- https://www.icoconverter.com/
- https://convertio.co/es/png-ico/

Por ahora, se usa un ícono por defecto de Electron si no existe este archivo.
